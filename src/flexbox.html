Flexbox images

The principle of layout with Flexbox is simple: you define a container, and inside you place several elements.
I used the following code:

#flex-wrap {
justify-content: space-evenly;
text-align: center;
flex-wrap: wrap;
display: flex;
}
#flex-wrap img {
max-width: 85%;
margin-top: 10px;
}
