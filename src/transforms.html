Transform scale

The scale() function allows you to modify the size of an element with a two-dimensional scale factor. This transformation is defined by a vector whose coordinates define the deformation applied in each direction.
I used the following code:

.vignette1 img {
transform: scale(0.5, 0.8);
}

Transform rotate

The rotate() function defines a transformation that moves an element around a fixed point without deforming it. It is a rotation around this point. By default, this origin corresponds to the center of the element.
I used the following code:

.vignette2 img {
transform: rotate(0.5turn);
}

Transform skew

The skew() function allows distortion by stretching each point of the element by a certain angle in a direction of the plane. To do this, the ordinate of an element is increased by a value proportional to the given angle and distance from the origin. The further away the point is from the origin, the greater the offset obtained.
I used the following code:

.vignette3 img {
padding: 2%;
transform: skew(10deg, 5deg);
}
