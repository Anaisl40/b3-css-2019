Responsive demo page

A site is "responsive" when its code meets standards that allow it to be adapted to all types of media (desktops, tablets, mobiles). Thus the size of the objects (especially the images in our example) adapts to the size of the screen where they are visible. It is a question of adapting the site to all resolutions. In other words, the content blocks are resized and reorganized according to the space available on the screen.

Flexbox demo page

FlexBox is a layout mode that provides a layout of the elements of a page in such a way that these elements have a predictable behavior when they have to accommodate different screen sizes. Many designers find the flexbox model easier to use. The child elements of a flexbox can be placed in any direction and can have dimensions to fit the available space. Positioning child elements is thus simpler and complex layouts can be set up more easily and with a cleaner code, since the order in which the elements are displayed is independent of their order in the source code.

Grid Layout demo page

The CSS Grid Layout module adds a two-dimensional grid to CSS. Grids can be used to arrange entire pages or small interface elements. A grid is a set of horizontal and vertical lines that intersect - the first defining the rows, and the second the columns. The elements are placed on the grid according to these rows and columns.

Transform demo page

The transform property modifies the coordinate space used for visual formatting. Thanks to this property, it is possible to translate the elements, rotate them, apply homotheties, distort them to change their perspective.
