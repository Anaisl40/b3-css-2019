Responsive images pages

Content images

The object-fit CSS property defines how the content of a replaced element should adapt to its parent using its width and height.
Here I used the following code:

#content-images img{
max-height: 200px;
object-fit: cover;
width: 100%;
}

Background image

The background-image property allows you to define one or more images as background for an element.
Here I used the following code:

#content-images2{
background-image: url('../gorilla.jpg');
background-repeat: no-repeat;
background-size: cover;
min-height: 400px;
background-position: center;
}
